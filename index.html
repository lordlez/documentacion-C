Introducción

Uno de los objetivos de diseño del lenguaje C es que solo sean necesarias unas pocas instrucciones en lenguaje máquina para traducir cada elemento del lenguaje, sin que haga falta un soporte intenso en tiempo de ejecución. Es muy posible escribir C a bajo nivel de abstracción; de hecho, C se usó como intermediario entre diferentes lenguajes. En parte, a causa de ser de relativamente bajo nivel y de tener un modesto conjunto de características, se pueden desarrollar compiladores de C fácilmente. En consecuencia, el lenguaje C está disponible en un amplio abanico de plataformas (más que cualquier otro lenguaje). Además, a pesar de su naturaleza de bajo nivel, el lenguaje se desarrolló para incentivar la programación independiente de la máquina. Un programa escrito cumpliendo los estándares e intentando que sea portátil puede compilarse en muchos computadores. C se desarrolló originalmente (conjuntamente con el sistema operativo Unix, con el que ha estado asociado mucho tiempo) por programadores para programadores. Sin embargo, ha alcanzado una popularidad enorme, y se ha usado en contextos muy alejados de la programación de software de sistema, para la que se diseñó originalmente.

Variables

El lenguaje C obliga a declarar una variable antes de ser usada. Declarar una variable no significa que se le asigne contenido, sino simplemente se indica que la variable existe. ¿Qué nos mostrará si intentamos mostrar o utilizar la variable si no le hemos asignado contenido? Hay lenguajes que asignan automáticamente contenido cero, vacío o falso a una variable, dependiendo de qué tipo sea. Sin embargo en C no está permitido usar una variable sin antes haberle asignado un contenido, a lo que se denomina “inicializar la variable”. Por tanto, al tratar de ejecutar un programa donde una variable está sin inicializar puede producirse un error de compilación. Además, las variables se pueden clasificar en dos grupos, globales y locales. Para poder utilizar variables es necesario especificar el tipo de dato que tendrá dicha variable, estos son algunos ejemplos de la declaración de variables:

int numero; float monedas; char nombre;

Sin embargo, aún nos falta asignarle algún valor a dichas variables. Los valores que les asignemos deben corresponder si o si al tipo de dato que la variable fue declarada. Por lo tanto, si queremos inicializar nuestras variables lo podriamos hacer de la siguiente manera:

numero = 5; monedas = 50,25; nombre = "Lorenzo";

Si prestás atención, podés ver que ya no hace falta poner el tipo de dato que tiene la variable, puesto que eso ya fue hecho en la declaración de la misma. Simplemente basta con asignarle un valor a la variable que corresponda con su tipo de dato y listo, ya tendriamos nuestras variables declaradas e inicializadas.

Condicionales

Los condicionales son sentencias de control donde la condición es la expresión que será evaluada. Si es verdadera el extracto se ejecuta, si es falsa, entonces el extracto es ignorado. Este tipo de sentencias permiten variar el flujo del programa en base a unas determinadas condiciones. Podemos diferencias la estructura if - else y la estructura switch. Comencemos con la primera:

if (condición){ sentencia1; }else{ sentencia2; }

Mientras que switch:

switch(variable){ case contenido_variable1: sentencias; break; case contenido_variable2: sentencias; break; defaulf: sentencias; }

Cada case puede incluir una o más sentencias sin necesidad de ir entre llaves, ya que se pueden ejecutar todas hasta que se encuentra la sentencia break. La variable evaulada solo puede ser de tipo entero o caracter. Default ejecutará las sentencias que incluya, en caso de que la opción escogida no exista.

Ciclos

Los ciclos son una estructura de control para repetir una o mas instrucciones. La ejecución del cuerpo del ciclo es controlada por una expresión lógica. En C contamos con tres tipos de cilos que podemos utilizar, los cuales son:

  • for
  • while
  • do while

El ciclo for, se caracteriza por su condicion de fin, el cual es previamente conocido al momento de ejecutarlo, y su sintáxis es la siguiente:

for(i = valor; i < condicion; i++){ sentencias; }

El ciclo while, por el contrario al ciclo for, no sabemos exactamente el fin previamente del ciclo de forma que se le debe dar una condicion de fin que, llegado el momento de cumplirse, el ciclo termina. Su sintaxis es asi:

while(condicion){ sentencia; }

Por ultimo, el ciclo do while, es un ciclo que se ejecutará por lo menos una vez. Es muy útil para crear menús, validación de datos, etc. Su sintáxis es la siguiente:

do{ sentencias; }while(condición);
Funciones

En C, se conocen como funciones aquellos trozos de códigos utilizados para dividir un programa con el objetivo que, cada bloque realice una tarea determinada. Una función puede ser llamada desde cualquier parte del programa no sólo una vez, y cuando es llamada, empieza a ejecutar las acciones que están escritas en código. Ejemplo de funciones:

  • Funciones que no devuelven ningún valor
  • Funciones que devuelven un valor entero
  • Funciones que devuelven un valor Real
  • Funciones combinadas
